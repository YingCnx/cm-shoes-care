# CM-SHOES-CARE

## วิธีใช้งาน

### Backend (Node.js + Express)
```
cd backend
npm install
npm start
```

### Frontend (React.js)
```
cd frontend
npm install
npm start
```

## โครงสร้างโปรเจค
- `backend/` → API Server
- `frontend/` → Web Dashboard
- `database/` → Data Storage
- `deploy/` → Deployment Scripts
